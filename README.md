# The Wordle Game
Guess the word in 7 attempts.

## Environment

- Node 16.14.0
- React 17.0.2

## Prerequisites

Node.js & npm

https://nodejs.org/en/download/

## Installation

Clone the project
```
$ git clone https://github.com/Anita9/wordle-game.git
$ cd wordle-game
```

Install dependencies
```
$ npm install
```

Run the development server
```
$ npm start
```

## Usage

Access the React app on your local machine

http://localhost:3000/
